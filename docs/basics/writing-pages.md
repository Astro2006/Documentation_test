## Writing Pages

This page describes a simple, consistent structure for Markdown docs pages.

## Suggested template

```md
## Page Title

One-sentence summary of what the page is for.

## Section

- Bullet points for key ideas
- Short paragraphs for details

## Related

- [Another page](./another-page.md)
```

## Tips

- Start headings at `##` to keep page titles consistent across the docs set
- Prefer shorter paragraphs for better scanning
- Add a “Related” section at the bottom for navigation
