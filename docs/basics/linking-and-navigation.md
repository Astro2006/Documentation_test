## Linking & Navigation

This page exists to test relative links across multiple directories.

## Relative link examples

- Same folder: `[FAQ](faq.md)`
- Child folder: `[Style Guide](../reference/style-guide.md)`
- Parent folder: `[Basics](../basics/basics.md)`

## Recommended patterns

- Prefer **relative links** so docs work across renderers
- Keep filenames stable (renames break links)
- Add “Related” links at the bottom of every page
