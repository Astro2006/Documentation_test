## Maintaining the TOC

This guide is for testing TOC maintenance workflows.

## Rules of thumb

- Every TOC entry should map to a real file path under `docs/`
- Keep ordering stable so changes are easy to review
- Use consistent section names and page titles

## Example review checklist

- [ ] All linked files exist
- [ ] Page titles match the TOC titles
- [ ] No duplicated “order” numbers within the same section
